# Landing Page - Agile Assistente Jurídico

Landing page profissional para o Agile, assistente de IA jurídico que funciona via WhatsApp.

## 🎨 Características

- **Design Dark Mode Elegante**: Tema escuro profissional com cores preto, cinza grafite e detalhes em azul neon
- **Totalmente Responsivo**: Adaptável para desktop, tablet e mobile
- **Copywriting Focado**: Textos diretos e profissionais voltados para advogados
- **Performance Otimizada**: Carregamento rápido e animações suaves

## 📁 Estrutura de Arquivos

```
├── index.html      # Estrutura HTML da landing page
├── styles.css      # Estilos CSS com tema dark mode
├── script.js       # JavaScript para interatividade
└── README.md       # Este arquivo
```

## 🖼️ Imagem do Robô

**Importante**: Você precisa adicionar a imagem do robô assistente jurídico com o nome `robot-assistant.jpg` na mesma pasta do `index.html`.

A imagem será exibida na Hero Section ao lado do título principal.

## 🚀 Como Usar

1. Adicione a imagem `robot-assistant.jpg` na pasta raiz
2. Abra o arquivo `index.html` em um navegador
3. Personalize os links do WhatsApp nos botões CTA (linhas 145-150 do index.html)

## ✏️ Personalização

### Links do WhatsApp
Edite os links nos botões CTA no arquivo `index.html`:
- Substitua `5511999999999` pelo número real do WhatsApp
- Personalize as mensagens pré-definidas se necessário

### Cores
As cores podem ser ajustadas nas variáveis CSS no início do arquivo `styles.css`:
- `--accent-blue`: Azul neon principal
- `--accent-gold`: Dourado para destaques
- `--bg-primary`: Cor de fundo principal

## 📱 Seções da Página

1. **Hero Section**: Título de impacto com imagem do robô
2. **Problema vs. Solução**: Contraste entre dificuldades e solução
3. **Funcionalidades**: 4 cards principais destacando recursos
4. **Prova Social**: Depoimentos e estatísticas
5. **CTA Final**: Chamada para ação principal
6. **Footer**: Links e informações legais

## 🎯 Copywriting

Todo o copy foi desenvolvido com foco em:
- Tom profissional e direto
- Benefícios claros para advogados
- Ênfase na simplicidade (WhatsApp)
- Linguagem técnica mas acessível
- Foco em produtividade e economia de tempo
